# Friend&Friends Wholesale Website

A professional B2B wholesale website built using React, Vite, and Tailwind CSS.

---

## 🚀 About the Project

Friend&Friends is a wholesale supply company focused on providing reliable products to businesses, resellers, and retailers.

This website is designed to:
- Showcase product catalogue
- Generate business inquiries
- Build trust with potential partners
- Allow users to send product requirements

---

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- TypeScript

---

## 📦 Features

- Clean and professional UI
- Product catalogue display (no pricing)
- “Request Quote” system
- Requirement form for custom product requests
- Responsive design (mobile + desktop)
- Smooth and fast performance

---

## ⚙️ Installation

Clone the repository:

git clone https://github.com/your-username/friendandfriends.git
cd friendandfriends

Install dependencies:

npm install

Run locally:

npm run dev

---

## 🌐 Deployment (Vercel)

1. Push code to GitHub  
2. Go to Vercel  
3. Import repository  
4. Click Deploy
