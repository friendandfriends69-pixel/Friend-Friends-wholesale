# Friend&Friends Wholesale Website (HTML Version)

A simple, lightweight, and professional B2B wholesale website built using only HTML, CSS, and JavaScript.

---

## 🚀 About

This website is designed for a wholesale supplier business:
- Show product catalogue
- Generate inquiries (no pricing shown)
- Allow users to send product requirements

---

## 📂 Project Structure

index.html → Main website structure  
style.css → Styling and layout  
script.js → Smooth scrolling & interactions  

---

## ⚡ Features

- Clean professional UI  
- No pricing display (B2B model)  
- “Request Price” buttons  
- Requirement form  
- Smooth scrolling  
- Lightweight (no frameworks)

---

## 🌐 Deployment (Vercel)

1. Upload these 3 files to GitHub  
2. Import repo in Vercel  
3. Click Deploy  

No configuration required.

---

## 📌 Note

This is a static website:
- No backend  
- No cart system  
- Focused on lead generation
